Pointer Arithmetic Example

A snippet demonstrating how pointer increments rely on data type size.

pointer_arith.c
// Demonstrating pointer arithmetic
int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int *ptr = arr;

    // Move pointer by 2 integers (8 bytes on 64-bit)
    ptr = ptr + 2;
    
    printf("%d\n", *ptr); // Prints 30
    
    return 0;
}
