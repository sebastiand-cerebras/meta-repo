Architecture & Components

main.py (Service Orchestrator)

The heart of the application. Uses FastAPI to serve endpoints. Handles the connection between the OpenAI LLM and the local SQLite database.

schemas.py

Pydantic models ensuring strict type validation for incoming requests and outgoing SQL results.

queries/

Directory containing SQL templates and raw query logic used for data ingestion.

Makefile

Automation for installation, data ingestion, running the server, and testing endpoints.
