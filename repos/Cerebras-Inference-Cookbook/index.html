Streaming Inference

Experience real-time token generation. The implementation uses Server-Sent Events (SSE) to push tokens as they are generated by the Cerebras cloud, removing the wait time for full completion.

  • Zero-copy data transfer
  • Automatic reconnection handling
  • Supported on WebSocket & HTTP
10x Faster than GPU
main.py
from cerebras_model import Client
# Initialize the client
client = Client(api_key=os.getenv())
# Stream response
for token in client.stream(prompt):
print(token, end="")
