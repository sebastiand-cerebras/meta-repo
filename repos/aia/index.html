Quick Start

Initialize your environment and run the core module.

# Import the core architecture
from aia import Core

# Initialize the architecture in adaptive mode
system = Core(mode='adaptive')

# Define your input data stream
input_data = ["source_a", "source_b"]

# Execute your workflow logic
result = system.run(input_data)

print(result)
